# App 01. 같이집중 — 레트로 집중 퀘스트 타이머

## 한 줄 소개

“해야 하는 일”을 작은 게임 퀘스트처럼 시작하게 도와주는 ADHD 친화형 집중 타이머 웹앱입니다.

## 왜 만들었나

집중이 어려운 사람에게 가장 힘든 순간은 종종 “집중을 유지하는 시간”보다 “시작 버튼을 누르기 전”입니다. 같이집중은 거창한 생산성 도구보다, 시작 장벽을 낮추는 귀여운 게임 콘솔 같은 경험을 목표로 만들었습니다.

## 핵심 경험

- 집중 시간과 쉬는 시간을 초·분·시 단위로 빠르게 설정합니다.
- 여러 세트를 반복하며 `FOCUS → BREAK → FOCUS` 흐름을 자동으로 이어갑니다.
- 마지막 5초에는 비프음이 울려서 작은 게임 카운트다운처럼 느껴집니다.
- 레트로 픽셀 UI, 선명한 상태 표시, 큰 타이머로 지금 해야 할 행동만 보이게 했습니다.

## 현재 구현된 기능

- 집중/휴식 시간 선택
- 빠른 시간 추가 버튼
- 반복 세트 수 설정
- 자동 단계 전환
- 시작/중단 컨트롤
- 완료 상태 표시
- 모바일에서도 사용 가능한 반응형 레이아웃

## 앞으로 확장할 기능

- 방 만들기와 참여 코드
- 잠금방 PIN
- 방 사람들끼리 “레디고!” 후 동시 타이머 시작
- 쉬는 시간 전용 채팅
- 딴짓 방지용 미니 장치
  - 타이머 중 화면 이탈 감지
  - 쉬는 시간 전까지 채팅 잠금
  - 캠 사용 시 얼굴을 직접 노출하지 않는 필터/아바타 모드
- 운동 타이머 프리셋
- 완료 기록과 작은 보상/뱃지

## 보안 관점에서 신경 쓴 점

이 앱은 포트폴리오용 공개 레포로 운영하되, 실제 서비스 확장 시에는 아래 원칙을 지킵니다.

- 시크릿/API 키는 코드에 넣지 않고 `.env`로 분리합니다.
- `.env`는 `.gitignore`에 포함합니다.
- 프론트엔드에는 공개 가능한 publishable key만 둡니다.
- 관리자 기능은 URL 숨기기가 아니라 인증/권한 체크로 보호합니다.
- 방, 참가자, 메시지는 UUID 기반 식별자를 사용합니다.
- 서버성 기능은 Supabase RLS/RPC 또는 백엔드에서 권한 검증 후 처리합니다.
- 입력값 검증, rate limit, 에러 메시지 최소화를 기본 정책으로 둡니다.

## 포트폴리오 포인트

이 프로젝트는 단순 타이머가 아니라 “집중 시작을 게임화하는 UX”를 실험한 첫 번째 앱입니다. 사용자가 스스로를 관리한다기보다, 작은 퀘스트를 하나씩 클리어하는 느낌을 주는 방향으로 발전시킬 예정입니다.
