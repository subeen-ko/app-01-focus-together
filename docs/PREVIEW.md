# Plan Game (레트로 인터벌 타이머)

## 한 줄 소개

“해야 하는 일”을 작은 아케이드 게임 퀘스트처럼 시작하게 도와주는 ADHD 친화형 인터벌(루프형) 타이머 웹앱입니다.

## 배포 링크

[https://app-01-focus-together.pages.dev](https://app-01-focus-together.pages.dev)

## 왜 만들었나

집중이 어려운 사람에게 가장 힘든 순간은 종종 “집중을 유지하는 시간”보다 “시작 버튼을 누르기 전”입니다. 거창한 생산성 도구보다, 일단 무지성으로 `START!` 버튼을 누르고 싶게 만드는 귀여운 레트로 게임 콘솔 같은 경험을 목표로 만들었습니다. 기존에 기획했던 방 만들기/멀티플레이 기능은 앱이 무거워지고 복잡해지는 원인이 되어 과감히 제거하고, 오직 '나 자신의 극한 집중'을 위한 타바타/뽀모도로 타이머 형태로 노선을 변경했습니다.

## 핵심 경험

- 실행 시간과 쉬는 시간을 직관적으로 분리해서 설정합니다.
- `+10초`, `+1분` 같은 오락실 버튼 느낌의 퀵 추가 기능으로 시간을 세팅합니다.
- 여러 세트를 반복하며 `FOCUS → BREAK → FOCUS` 흐름을 자동으로 이어갑니다.
- 5초 전부터 울리는 비프음과 0초에 울리는 완료음이 게임 카운트다운 같은 짜릿함을 줍니다.
- 레트로 픽셀 UI(네온 테두리, 깜빡이는 START 문자)로 집중력을 환기시킵니다.

## 현재 구현된 기능

- 직관적인 탭(실행 시간 / 쉬는 시간) 분리 및 퀵 타임 버튼 연동
- 반복 세트 수 조절 (`-` / `+` 버튼)
- State Machine 기반의 자동 페이즈(Phase) 전환 시스템
- 단계별 시각적 분리 (Focus: 핑크, Break: 민트, Clear: 옐로우)
- 모바일(iOS Safari 등)에서도 완벽하게 동작하는 오디오 해제 로직
- 모바일 화면에 꼭 맞는 반응형 픽셀 레이아웃

## 보안 및 아키텍처

- 현재 버전은 어떠한 외부 백엔드(Supabase 등)나 DB 연결 없이 **100% 브라우저(Client-Side) 환경에서만 독립적으로 동작**합니다.
- 개인정보나 세션 데이터를 저장하지 않아 보안상 매우 안전하며 가볍습니다.

## 기술 스택

- React
- Vite
- CSS (Vanilla, No-Tailwind)
- Cloudflare Pages

## 포트폴리오 포인트

이 프로젝트는 화려한 서버 연동보다 **"UX(사용자 경험)와 미세한 디테일(마이크로 인터랙션)"**에 극한으로 집중한 결과물입니다. 
1. `Date.now()`를 활용해 백그라운드 타이머의 Stale State 버그를 완벽히 잡았고,
2. 모바일 브라우저의 엄격한 오디오 자동 재생 정책을 뚫기 위해 무음(Silent) 오실레이터 핵을 적용했으며,
3. Tailwind 없이 순수 CSS만으로 8-bit 아케이드 감성의 네온 UI를 깎아냈습니다.
